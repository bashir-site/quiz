### Admin
- Создайте учетную запись администратора с помощью команды
```
py manage.py createsuperuser
```
- После входа в систему вы можете увидеть общее количество учеников, учителей, курсов, вопросы, которые есть в системе на панели инструментов.
- Может просматривать, обновлять, удалять, утверждать учителя.
- Может просматривать, обновлять, удалять ученика.
- Также можно увидеть оценки учащихся.
- Можно добавлять, просматривать, удалять курс / экзамены.
- Можно добавлять вопросы к соответствующим курсам с вариантами, правильным ответом и отметками.
- Также можно просматривать и удалять вопросы.

### Teacher
- Подать заявку на работу в Системе. Затем войдите в систему (требуется одобрение системного администратора, тогда только учитель может войти в систему).
- После входа в систему вы можете увидеть общее количество студентов, курс, вопросы, которые есть в системе на панели инструментов.
- Можно добавлять, просматривать, удалять курс / экзамены.
- Можно добавлять вопросы к соответствующим курсам с вариантами, правильным ответом и отметками.
- Также можно просматривать и удалять вопросы.
> **_NOTE:_**  Обычно администратор нанимает учителей для управления курсами и вопросами.

### Student
- Создать учетную запись (одобрение администратора не требуется, можно войти после регистрации)
- После входа в систему можно увидеть, сколько курсов / экзаменов и вопросов есть в системе на панели инструментов.
- Можно сдать экзамен в любое время, ограничения по количеству попыток нет.
- Может просматривать оценки каждой попытки каждого экзамена.
- Шаблон вопроса - MCQ с 4 вариантами и 1 правильным ответом.

## HOW TO RUN THIS PROJECT
- Установите Python (3.7.6) (не забудьте указать «Добавить в путь» при установке Python)
- Откройте терминал и выполните следующие команды:
```
python -m pip install -r requirements. txt
```
- Загрузите эту zip-папку проекта и распакуйте ее.
- Переместить в папку проекта в Терминале. Затем запустите следующие команды:
```
py manage.py makemigrations
py manage.py migrate
py manage.py runserver
```
- Теперь введите следующий URL-адрес в вашем браузере, установленном на вашем ПК.
```
http://127.0.0.1:8000/
```

## CHANGES REQUIRED FOR CONTACT US PAGE
- В файле sets.py вы должны указать свой адрес электронной почты и пароль.
```
EMAIL_HOST_USER = 'youremail@gmail.com'
EMAIL_HOST_PASSWORD = 'your email password'
EMAIL_RECEIVING_USER = 'youremail@gmail.com'
```

## Drawbacks/LoopHoles
- Администратор / преподаватель может добавить любое количество вопросов к любому курсу, но при добавлении курса администратор предоставляет номер вопроса.
